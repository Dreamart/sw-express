# SWCRM Application

DREAMART STUDIO PRESENTS

## Requirements

SWCRM is a Symfony 2 based application with the following requirements:

* PHP 5.4.9 or above with command line interface
* PHP Extensions:
    * GD
    * Mcrypt
    * JSON
    * ctype
    * Tokenizer
    * SimpleXML
    * PCRE
    * ICU
* MySQL 5.1 or above
* PostgreSQL 9.1 or above
